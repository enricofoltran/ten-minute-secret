F.A.Q.

Can I retrieve a secret that has already been viewed?

No. We display it once.

How long do you keep non-viewed secrets?

We keep secrets for 10 minutes.

What is the maximum secret size?

The maximum size is 50KB.

Why should I trust you?

We don't store the passphrase so we can never know what the secret is because we can't decrypt it.

The code is open-source so you can review it and/or run your own instance.
